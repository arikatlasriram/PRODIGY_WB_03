# Tic-Tac-Toe Game 🎮

A simple and interactive Tic-Tac-Toe web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Two player gameplay
- Interactive UI
- Winner detection
- Draw match detection
- Restart game option
- Responsive design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
```bash
├── index.html
├── style.css
├── script.js
```

## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in your browser
